# ML-Customer_churn

# 📊 **Customer Churn Prediction Project**

A complete machine learning project to analyze customer behavior and predict which customers are likely to leave (churn).
This repository includes data preprocessing, EDA, visualization, model training, evaluation, and prediction.

---

## ⭐ **Project Highlights**

* 📁 **Realistic business dataset**
* 📈 **Full Exploratory Data Analysis (EDA)**
* 🤖 **Machine Learning models for prediction**
* 🧹 **Clean and consistent preprocessing pipeline**
* 🎨 **Clear charts & visual storytelling**
* 🔍 **Interpretation of model performance**
* 🚀 **Easy to run and customize**

## 🎯 **Objectives**

* Understand patterns of customer churn
* Build a machine learning model to predict churn
* Identify key factors influencing customer departures
* Provide actionable insights for customer retention

---

## 🧠 **Machine Learning Workflow**

### 🗃️ 1. Data Loading

* Import CSV dataset
* Check columns, data types, and missing values

### 🔍 2. Exploratory Data Analysis

Includes:
📊 Correlation heatmaps
📉 Churn distribution
👥 Customer demographic breakdown
💳 Behavior patterns, activity level, account info

### 🧹 3. Preprocessing

* Encoding categorical features
* Scaling numerical features
* Handling class imbalance (SMOTE)
* Train-test split

### 🤖 4. Model Training

Models used:

* Logistic Regression
* Random Forest
* XGBoost
* Decision Tree
* KNN

### 📏 5. Model Evaluation

Metrics:

* Accuracy
* Precision
* Recall
* F1-Score
* ROC-AUC

### 🔮 6. Prediction

Predict churn probability for new customers.
